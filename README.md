## Installation

Run the following

```bash
# Clone the repository
git clone https://github.com/Harshal12355/ISKCON-LLM.git

# Navigate to the project directory
cd ISKCON-LLM

# Install dependencies
pip install -r ./requirements.txt
# also ensure you have pytorch installed
```
